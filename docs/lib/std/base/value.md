## value
Константное значение.<br>

### Применение
1. `(value 12)`<br>
`12` - цифровая константа.<br><br>
2. `(value "Text")`<br>
`text` - строчная константа.<br><br>
3. `(value ^java.lang.Object)`<br>
`java.lang.Object` - константа в виде класса.<br><br>

### Примеры

```pihta
(def [[j (value 21)]])
```

## field
Объявляет `переменные`.

### Применение

1. `(def [[^int i]])`<br>
`[^int i]` - _переменная_.<br>
|`^int` - _тип_.<br>
|`i` - _имя_.<br><br>
2. `(def [[j 12]])`<br>
`[i 12]` - _переменная_.<br>
|`j` - _имя_.<br>
|`12` - _значение_.<br><br>
3. `(def [[^long k 21l]])`<br>
`[^long k 21l]`<br>
|`^long` - _тип_.<br>
|`k` - _имя_.<br>
|`21l` - _значение_.

<h4>Примеры</h4>

```pihta
(def [[^int i]])
```

```pihta
(def [[j 12]])
```

```pihta
(def [[^long k 21l]])
```

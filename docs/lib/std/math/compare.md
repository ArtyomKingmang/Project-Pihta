## <=>
Сравнивает 2 `значения` `выражений`.

- Результат `<` 0 - меньше.
- Результат `=` 0 - равно.
- Результат `>` 0 - больше.

### Применение

1. `(<=> (expr0) (expr1))`<br>
`(expr0)` `(expr1)` - _выражения_.

### Примеры

```pihta
(use std/base std/math)
(#println std
    (<=> 1 2)
    (<=> 3 3)
    (<=> 5 4))
```
